# API Test Tribal MNC

## Stack principal
- **Python 3.7+**
- **FastAPI**

## Instalaci√≥n
  1. Instalar Python 3.7+ desde https://www.python.org/downloads/ o usando los paquetes del sistema
      ```console
     $ apt install python3.7
      ```
  2. Clonar este repositorio en la ubicacion preferida
      ```console
     $ git clone https://github.com/maneul198/api-test-tribal
      ```
  3. Entrar en la carpeta del proyecto
      ```console
     $ cd api-test-tribal 
      ```
  4. Crear un entorno virtual para las dependencias
      ```console
     $ python3 -m venv env 
      ```
  5. Activar entorno virtual
      ```console
     $ source env/bin/activate
      ```
  6. Instalar las dependencias necesarias para correr el proyecto
      ```console
     $ pip install -r requirements.txt
      ```
 
 ### **Correr el proyecto**
 ```console
 $ python main.py
 ```
##### Ahora abre en el navegador el siguiente enlace: http://localhost:8000/docs
Debes encontrar la documentacion necesaria para interactuar con el API
